1. HTML Introduction

HTML (HyperText Markup Language) est le langage de base pour créer des pages web. Il structure le contenu et permet de l'afficher dans un navigateur.

Structure minimale d'une page :

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="utf-8">
  <title>Ma page</title>
</head>
<body>
  <h1>Bonjour</h1>
</body>
</html>

Explication : la déclaration <!DOCTYPE html> indique que vous utilisez HTML5.

2. HTML Basic

Les balises de base :

  • <html> : racine du document
  • <head> : méta-infos, titre, fichiers CSS/JS
  • <title> : nom de la page dans l'onglet
  • <body> : contenu visible
<html>
  <head><title>Titre</title></head>
  <body>Contenu</body>
</html>

Conseil : sauvegarde toujours en UTF-8 et utilise des balises sémantiques quand possible.

3. HTML Elements

Un élément HTML se compose d'une balise ouvrante, du contenu, et d'une balise fermante :

<p>Ceci est un paragraphe</p>
<a href="https://example.com">Lien</a>
<img src="image.jpg" alt="Description">

Note : certaines balises sont auto-fermantes (ex : <img>).

4. HTML Attributes

Les attributs fournissent des informations additionnelles aux éléments :

<a href="https://example.com" target="_blank" rel="noopener">Visiter</a>
<img src="logo.png" alt="Logo" width="200">

Bonnes pratiques : toujours fournir un alt pour les images pour l'accessibilité.

5. HTML Headings

Utilise <h1> à <h6> pour structurer le contenu. Le SEO se base sur cette hiérarchie.

<h1>Titre principal</h1>
<h2>Sous-titre</h2>

Conseil : un seul <h1> par page pour la structure sémantique.

6. HTML Paragraphs

Les paragraphes sont créés avec <p>. Pour la mise en forme : <strong>, <em>, etc.

<p>Texte normal</p>
<strong>Texte en gras</strong>
<em>Texte en italique</em>

7. HTML Links

Liens internes vs externes :

<a href="page.html">Lien interne</a>
<a href="https://example.com" target="_blank" rel="noopener">Lien externe</a>

Pour naviguer dans la même page, on utilise des ancres : <a href="#section">.

8. HTML Images

Insérer une image correctement :

<img src="photo.jpg" alt="Description claire" width="400">

Astuce : optimise la taille pour la performance et fournis des srcset si nécessaire.

9. HTML Lists

Listes non-ordonnées et ordonnées :

<ul>
  <li>Point 1</li>
  <li>Point 2</li>
</ul>

<ol>
  <li>Premier</li>
  <li>Deuxième</li>
</ol>

10. HTML Tables

Tables pour afficher des données tabulaires :

<table border="1">
  <tr><th>Nom</th><th>Âge</th></tr>
  <tr><td>Alice</td><td>25</td></tr>
</table>

Pour le style, préfère CSS plutôt que l'attribut border inline.

11. HTML Forms

Les formulaires collectent des données utilisateur et les envoient au serveur.

<form action="/submit" method="post">
  <label for="nom">Nom:</label>
  <input id="nom" name="nom" type="text" required>
  <input type="submit" value="Envoyer">
</form>

Pense à la sécurité (validation côté serveur) et à l'accessibilité (labels).

12. HTML Input Types

Types d'input courants :

<input type="text">  <!-- texte -->
<input type="email"> <!-- email -->
<input type="number"> <!-- nombre -->
<input type="date">   <!-- date -->

Utilise le bon type pour activer les claviers mobiles adaptés et la validation native.

13. HTML Buttons

Boutons :

<button type="button">Cliquer</button>
<button type="submit">Envoyer</button>
<input type="reset" value="Réinitialiser">

14. HTML Select & Options

Menu déroulant :

<label>Pays:</label>
<select name="pays">
  <option value="fr">France</option>
  <option value="us">USA</option>
</select>

15. HTML Textarea

Zone de texte multi-lignes :

<label for="msg">Message:</label>
<textarea id="msg" rows="6" cols="60">Votre message...</textarea>

16. HTML Multimedia

Inclure audio / vidéo :

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls width="640">
  <source src="video.mp4" type="video/mp4">
</video>

17. HTML Iframes

Afficher une page externe :

<iframe src="https://example.com" width="800" height="450" title="Exemple"></iframe>

Attention à la sécurité (sandbox) et au responsive.

18. HTML Semantic Elements

Balises sémantiques : <header>, <nav>, <main>, <article>, <aside>, <footer>.

<header>...</header>
<nav>...</nav>
<main>...</main>

19. HTML Meta Tags

Méta-informations pour le SEO et le comportement :

<meta charset="utf-8">
<meta name="description" content="Description de la page">
<meta name="viewport" content="width=device-width,initial-scale=1">

20. HTML Comments

Commentaires invisibles dans le rendu :

<!-- Ceci est un commentaire -->

21. HTML Entities

Caractères spéciaux et entités :

&lt; &gt; &amp; &copy; &nbsp;

22. HTML Colors

Définir les couleurs avec noms, hex ou rgb :

<p style="color:red">Texte rouge</p>
<p style="color:#007BFF">Texte bleu</p>

23. HTML Inline CSS

Styles en ligne (à limiter) :

<div style="background:#e6f0ff;padding:10px">Contenu</div>

24. HTML Internal CSS

Styles définis dans <style> dans le head :

<style>
p{color:green}
</style>

25. HTML External CSS

Relier un fichier CSS externe :

<link rel="stylesheet" href="style.css">

Pratique pour séparer la présentation du contenu.

26. HTML Classes & IDs

Utilise des classes pour le style et les IDs pour les identifiants uniques :

<div id="hero"></div>
<p class="important">Texte</p>

27. HTML Div & Span

<div> pour les blocs, <span> pour le texte en ligne.

<div class="card"><span>Label</span></div>

28. HTML Layouts

Organiser la page: header, nav, main, aside, footer. Combine avec CSS Grid / Flexbox pour les layout avancés.

<header></header>
<nav></nav>
<main><section></section></main>
<aside></aside>
<footer></footer>

29. HTML Tables Advanced

Fusion de cellules et en-têtes :

<table>
  <tr><th>Nom</th><th>Détails</th></tr>
  <tr><td colspan="2">Titre fusionné</td></tr>
</table>

30. HTML Best Practices

Règles recommandées :

  • Code sémantique et accessible
  • Utilise des balises ARIA si nécessaire
  • Valide ton HTML et CSS
  • Optimise les images et les ressources
  • Commentaires et structure claire
<!-- Ex : header sémantique et nav clair -->
<header><nav>...</nav></header>
